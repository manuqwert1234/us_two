# Us Two 💕

**A beautiful, thoughtful couple's cycle tracking app** — built to bring partners closer through understanding and sync.

**"Sync your cycles. Strengthen your connection."**


## ✨ Features

### For Her (Girlfriend)
- **One-tap period & symptom logging** (Cramps, Mood swings, Breast tenderness, Ovulation signs, etc.)
- **Smart cycle predictions** using actual average cycle length from history (not fake 28-day assumption)
- **Ovulation window detection** with boyfriend-friendly explanations
- **Ghost period reminders** — gently nudges if a period is expected but not logged
- **Past period history** for more accurate long-term predictions

### For Him (Boyfriend)
- Real-time visibility into her current cycle phase
- Clear, non-overwhelming insights and gentle tips on how to support her
- Sees logged symptoms instantly
- Builds empathy through transparency and shared understanding

### Shared
- **Unique Couple Code** system (e.g. `ABCD-12`)
- Firebase-powered real-time sync
- Anonymous authentication (no email required)
- Beautiful, calming turquoise + mint design
- Works on Android & iOS

## 🛠 Tech Stack

- **Flutter** (Dart)
- **Firebase** (Auth + Firestore)
- **Shared Preferences** (local persistence)
- **Table Calendar**
- **Google Fonts**
- **Shorebird** (for easy over-the-air updates)
